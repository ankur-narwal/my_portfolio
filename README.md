# my_portfolio
This is a personal portfolio website designed to showcase my skills, projects, achievements, and experience as a Computer Science student and developer. The site uses a scroll-based, section-reveal design to create an engaging and interactive browsing experience.

Features - 

 - Unique scroll-down layout that reveals sections dynamically.
 - Showcases 3 major projects, skills, and blogs.
 - Responsive design optimized for desktop and mobile devices.
 - Minimal, professional theme with smooth navigation.

Tech Stack - 

 - Front-End: HTML, CSS, JS
 - Animations & Effects: CSS transitions and scroll-based animations

Key Highlights - 

 - Highlights all major academic and technical accomplishments in a clean, professional format.
 - Uses a scroll-based storytelling design to keep visitors engaged.
 - Fully responsive layout for seamless viewing across all screen sizes.
